Anuario Ornitológico de Albacete Online

anuario.albacete.org

Aplicación web desarrollada para la Sociedad Albacetense de Ornitología (SAO) que recoge y publica los datos de avistamientos de aves en la provincia de Albacete para su posterior estudio con la finalidad principal de proteger las especies amenazadas. Permite registro de usuarios para que puedan añadir citas online geolocalizadas con imágenes asociadas.

  • Periodo de desarrollo: Entre el 2012 y el 2014

  • Tecnologías utilizadas para su desarrollo: PHP con el framework CakePHP, base de datos MySQL, HTML5, CSS3, Javascript, Bootstrap.
